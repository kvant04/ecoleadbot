# Перечни природоохранной документации по категориям объектов НВОС (справочник ЭкоЧекАп)

> **Тип документа:** Справочник перечней документов по категориям НВОС | **Номер:** NVOS-CAT-REF | **Дата принятия:** —

Ориентир для сверки полноты комплекта природоохранной документации по категориям объектов НВОС (I–IV). Подготовлен для EcoLeadBot RAG Assistant и отчётов ЭкоЧекАп. Не заменяет официальные нормативные акты.

## Warnings

- Справочный ориентир для сверки комплекта природоохранной документации; не заменяет официальные нормативные акты.
- Номера приказов, формы отчётности и сроки подачи — уточнять по действующим редакциям НПА (КонсультантПлюс / pravo.gov.ru).
- При использовании в отчёте ЭкоЧекАп не смешивать перечни категорий I–IV (КЭР, ДВОС и иные документы относятся к разным категориям).

## Текст документа

Назначение: ориентир для сверки полноты комплекта документов при подготовке раздела 6 отчёта ЭкоЧекАп. **Не заменяет официальные нормативные акты**; актуальные требования и реквизиты приказов уточнять по действующей редакции НПА.

**Общее для любой категории НВОС (если применимо по деятельности):** паспорта отходов I–IV классов опасности; при статусе производителя/импортёра товаров и/или упаковки — отчётность по экологическому сбору (условия и сроки — по НПА).

## I категория НВОС

**Для отчёта ЭкоЧекАп:** ориентирный перечень для **I** строится вокруг **КЭР**. **ДВОС** в том виде, как в блоке **II категории** ниже, для сверки **I** **не** используйте и **не** фиксируйте «ДВОС не представлена» как пробел по перечню II.

Типовой ориентирный перечень (сроки действия и обновления — по КЭР и специальным нормам; сверять с материалами объекта):

| Наименование | Срок / периодичность (ориентир) |
|--------------|----------------------------------|
| Комплексное экологическое разрешение (КЭР) | 7 лет |
| Инвентаризация источников выбросов; расчёт и утверждение нормативов допустимого воздействия (НДВ) | В составе КЭР; не позднее чем через 2 года после ввода в эксплуатацию при наличии разрешения на ввод / иначе по правилам ведения учёта |
| Технологические нормативы (выбросы, сбросы, допустимые физические воздействия) | На период действия КЭР |
| Проект нормативов образования отходов и лимитов на размещение (ПНООЛР) | В составе КЭР, на период КЭР |
| Расчёт нормативов допустимых сбросов | В составе КЭР при наличии сбросов |
| Программа производственного экологического контроля (ПЭК) | Ведение и актуализация при изменении процесса |
| Программа повышения экологической эффективности | При невозможности соблюдения нормативов; план мероприятий по ООС — ориентир до 7 лет, без продления |
| План мероприятий по уменьшению выбросов в периоды неблагоприятных метеорологических условий (ПНМУ) | По итогам инвентаризации; актуализация при изменении процесса |
| Свидетельство / выписка о постановке на учёт объекта НВОС, актуализация, снятие с учёта | По учётным правилам |
| Паспорта отходов | На образовавшиеся отходы I–IV кл. |
| Журналы учёта движения отходов | Квартально (накопительно) и годовой обобщённый |

Дополнительно по отчётности: декларация о плате за НВОС; формы 2-ТП (отходы, воздух, водхоз) — при выполнении критериев подачи; отчёт в региональный кадастр отходов — при обращении с отходах (региональные сроки).

## II категория НВОС

**Для отчёта ЭкоЧекАп:** **ДВОС** (декларация о воздействии на окружающую среду) в этом справочнике отнесена к ориентиру **II категории**. Для объектов **I, III, IV** категории **не** требуйте в §6 отчёта строку «ДВОС не представлена» по правилам этого блока.

Ориентирный перечень документов и отчётов (обзорная структура для практической сверки).

### Оглавление перечня

- Декларация о воздействии на окружающую среду (ДВОС).
- Нормативы допустимых выбросов (НДВ) загрязняющих веществ в атмосферный воздух.
- Отчёт об инвентаризации источников выбросов загрязняющих веществ в атмосферный воздух.
- Нормативы допустимых сбросов (НДС) загрязняющих веществ в водный объект.
- Проект нормативов образования отходов и лимиты на их размещение (ПНООЛР).
- План мероприятий по охране окружающей среды.
- План мероприятий по уменьшению выбросов загрязняющих веществ в атмосферный воздух в периоды неблагоприятных метеорологических условий (НМУ).
- Программа производственного экологического контроля (ПЭК).
- Декларация о плате за негативное воздействие на окружающую среду (НВОС).
- Статистический отчёт по форме № 2-ОС.
- Статистический отчёт по форме № 4-ОС.
- Статистический отчёт по форме 2-ТП воздух.
- Статистический отчёт № 2-ТП водхоз.
- Статистический отчёт № 2-ТП отходы.
- Отчёт в региональный кадастр отходов.
- Журнал учёта движения отходов.

### Пояснения по позициям

**Декларация о воздействии на окружающую среду (ДВОС).**
Форма декларации утверждена Приказом Минприроды № 509 от 11.10.2018 г. ДВОС подаётся в контролирующие органы один раз в 7 лет при неизменности технологического процесса на объекте.

*Примечание:* объект II категории НВОС может претендовать на получение КЭР при наличии соответствующего отраслевого справочника наилучших доступных технологий (НДТ).

**Нормативы допустимых выбросов (НДВ)** загрязняющих веществ в атмосферный воздух.
Являются частью ДВОС, но необходимо разработать отдельный проект НДВ по Приказу Минприроды № 581 от 11.08.2020 г. и получить санитарно-эпидемиологическое заключение (СЭЗ). Только после этого включить рассчитанные нормативы в ДВОС.

**Отчёт об инвентаризации источников выбросов** загрязняющих веществ в атмосферный воздух.
Порядок проведения инвентаризации и правила разработки отчёта утверждены Приказом Минприроды № 352 от 07.08.2018 г.

**Нормативы допустимых сбросов (НДС)** загрязняющих веществ в водный объект.
Являются обязательной частью ДВОС, отдельного согласования не требуется. НДС рассчитываются согласно Приказу № 1118 от 29.12.2020 г.

**Проект нормативов образования отходов и лимиты на их размещение (ПНООЛР).**
Рассчитанные нормативы и лимиты включены в состав ДВОС. Отдельное согласование ПНООЛР в контролирующих органах не требуется, но наличие отдельного проекта в составе природоохранной документации предприятия обязательно; при проверке он может быть запрошен даже при наличии утверждённой ДВОС. Порядок разработки утверждён Приказом № 1029 от 08.12.2020 г.

**План мероприятий по охране окружающей среды.**
Разрабатывается при невозможности соблюдения нормативов допустимых выбросов и сбросов загрязняющих веществ, включённых в перечень загрязняющих веществ, в отношении которых применяются меры государственного регулирования в области охраны ОС (перечень — в Распоряжении Правительства РФ № 1316-р от 08.07.2015 г.). Правила разработки плана — Приказ Минприроды № 667 от 17.12.2018 г.

**План мероприятий по уменьшению выбросов** загрязняющих веществ в атмосферный воздух в периоды неблагоприятных метеорологических условий (НМУ).
Разрабатывается на основе проекта НДВ. Требуется объектам, имеющим источники выбросов. Правила разработки утверждены Приказом Минприроды № 811 от 28.11.2019 г.

**Программа производственного экологического контроля (ПЭК).**
Разрабатывается на основании Приказа Минприроды № 74 от 28.02.2018 г. По результатам контроля ежегодно разрабатывается и подаётся отчёт до 25 марта года, следующего за отчётным. Форма отчёта утверждена Приказом Минприроды № 261 от 14.06.2018 г.

**Декларация о плате за негативное воздействие на окружающую среду (НВОС).**
За выбросы, сбросы и размещение отходов предоставляется ежегодно до 10 марта года, следующего за отчётным. Плата за НВОС — до 1 марта. Форма декларации утверждена Приказом Минприроды № 3 от 09.01.2017 г.

**Статистический отчёт по форме № 2-ОС.**
При использовании водных объектов. Ежегодно до 25 января года, следующего за отчётным. Форма — Приказ Росстата № 469 от 28.08.2012 г.

**Статистический отчёт по форме № 4-ОС.**
Если на объекте осуществляется природоохранная деятельность. Ежегодно до 25 января года, следующего за отчётным. Форма — Приказ Росстата № 399 от 21.07.2020 г.

**Статистический отчёт по форме 2-ТП воздух.**
Если выбросы при наличии веществ 1 и 2 классов опасности превышают 5 т/год, либо выбросы свыше 10 т/год независимо от класса опасности. Ежегодно до 22 января года, следующего за отчётным. Форма — Приказ № 661 от 08.11.2018 г.

**Статистический отчёт № 2-ТП водхоз.**
При сбросах загрязняющих веществ в водные объекты либо заборе из водных объектов свыше 50 м³/сутки. Ежегодно до 22 января года, следующего за отчётным. Форма — Приказ Росстата № 815 от 27.12.2019 г.

**Статистический отчёт № 2-ТП отходы.**
При обращении с отходами (сбор, накопление, транспортирование, обработка, утилизация, обезвреживание, размещение). Ежегодно до 1 февраля года, следующего за отчётным. Форма — Приказ Росстата № 627 от 09.10.2020 г.

**Отчёт в региональный кадастр отходов.**
При деятельности по обращению с отходами. В зависимости от региона — ежеквартально или ежегодно. Форма — акт регионального органа.

**Журнал учёта движения отходов.**
Формируется ежеквартально; в конце года — обобщённый годовой журнал. Форма — Приказ Минприроды России № 1028 от 08.12.2020 г.

**Независимо от категории:** паспорта на отходы I–IV классов; отчёты по экологическому сбору — для производителей/импортёров товаров и/или упаковки (ежегодно, условия по НПА).

## III категория НВОС

**Для отчёта ЭкоЧекАп:** перечень ниже — **только** для объектов **III категории**. **Не** смешивать с перечнем **I категории**: **КЭР** в типовом ориентире **III не входит**. **Не** смешивать с перечнем **II**: **ДВОС** как обязательный элемент перечня II для III **не** подставлять; писать «КЭР не представлено» или «ДВОС не представлена» для III по правилам I/II **ошибочно**.

Ориентирный перечень (условия применимости — по фактической деятельности и НПА):

- Отчёт об инвентаризации источников выбросов загрязняющих веществ в атмосферный воздух (Приказ Минприроды № 871 от 19.11.2021 г. — актуализировать по действующей редакции).
- Отчёт об инвентаризации отходов производства и потребления — не всегда обязателен; при сложной технологии для определения перечня отходов.
- Проект нормативов допустимых выбросов (НДВ) — при наличии в выбросах веществ I и II классов опасности (Приказ Минприроды № 581 от 11.08.2020 г.; СЭЗ в Роспотребнадзоре).
- Проект нормативов допустимых сбросов (НДС) — при сбросах веществ I и II класса (Приказ № 1118 от 29.12.2020 г.).
- План мероприятий по уменьшению выбросов в периоды НМУ (Приказ Минприроды № 811 от 28.11.2019 г.; региональные НПА возможны).
- Программа ПЭК и ежегодный отчёт о ПЭК (Приказ Минприроды № 109 от 18.02.2022 г.; отчёт до 25 марта; форма — Приказ № 522 от 16.10.2018 г. — уточнять редакции).
- Декларация о плате за НВОС (форма Приказа № 1043 от 10.12.2020 г.; подача до 10 марта, плата до 1 марта — ориентир по типовым срокам).
- Форма 2-ОС — при использовании водных объектов (до 25 января; Приказ Росстата № 469).
- Форма 4-ОС — при затратах на охрану ОС свыше порога по статистике (Приказ Росстата № 530 от 29.07.2022 и др. — уточнять).
- 2-ТП отходы — при обращении с отходами (до 1 февраля; Приказ Росстата № 627).
- 2-ТП воздух — при превышении порогов по массе выбросов и классам опасности (до 22 января; Приказ № 661).
- 2-ТП водхоз — при сбросах в водные объекты или заборе > 50 м³/сутки (до 22 января; Приказ Росстата № 815).
- Отчёт в региональный кадастр отходов — при обращении с отходами (периодичность — по региону).
- Журнал учёта движения отходов (Приказ № 1028 от 08.12.2020 г.; кварталы и год).
- План мероприятий по охране ОС при временно разрешённых выбросах/сбросах — ориентир срока реализации до 7 лет без продления.

Учёт объекта НВОС (выписка, регистрация) — по общим правилам.

## IV категория НВОС

**Для отчёта ЭкоЧекАп:** перечень ниже — **только** для **IV категории**; **КЭР** и **ДВОС** (в роли обязательного документа перечня II) **не входят** в типовый ориентир IV. Не смешивать с перечнями I и II.

Ориентирный минимальный перечень:

- Отчёт / сведения об инвентаризации источников выбросов загрязняющих веществ в атмосферный воздух (порядок — по действующим приказам Минприроды по инвентаризации выбросов).
- Статистический отчёт 2-ТП отходы — при обращении с отходами (сбор, накопление, транспортирование, обработка, утилизация, обезвреживание, размещение); сроки и форма — по Росстату (ориентир: Приказ № 627).
- Отчёт в региональный кадастр отходов — при обращении с отходами (периодичность по региону).
- Журнал учёта движения отходов (квартальный и годовой; Приказ № 1028).

Учёт НВОС; паспорта отходов I–IV классов; при необходимости — экологический сбор, декларация платы за НВОС и иная отчётность по условиям деятельности (как для старших категорий).

**Независимо от категории:** паспорта на отходы I–IV классов; экологический сбор для производителей/импортёров товаров и упаковки.
